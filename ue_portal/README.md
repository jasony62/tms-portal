# ue

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dll(项目开发环境采用了DllPlugin，故启动前需要先执行，生成独立的第三方库)
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
